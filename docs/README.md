# awesome-porto

> A guide aiming to help everyone to have a successful start in Porto!

# Get Started

Are you coming to (or just arrived) in Porto? Awesome! In the table below you will find a summary (and recommendation!) of the first tasks that you might need to do in order to have a great start in this country. For more information, please read the rest of this guide.
